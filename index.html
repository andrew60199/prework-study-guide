HTML

  • The head element contains information about the webpage.
  • Inside the head element is a container for metadata.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS can be achieved in a number of ways, to ensure Separation of concerns the code is done in its own css file and linked to in the head of the html file

Git

  • pwd: Displays the current working directory.
  • ls: Displays the list of files and folders in the current directory.
  • cd: Navigates between directories in the path.
  • cd ..: Navigates backwards one level at a time, through the path of your directories.
  • mkdir: Creates a new directory.
  • touch: Creates a file inside a directory.
  • start: Opens a file.
  • git status: Checks what branch we are currently on.
  • git checkout -b branch-name: Creates a new branch and switches to it
  • git checkout main: Takes you to the main branch.
  • git add -A: Stage your changes
  • git commit -m "": adds a message to the commit
  • git pull origin main: takes you to the main branch
  • git push origin ***: push the branch to github to be reviewed and hopefully merged

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions are the building blocks of JavaScript.
  • That must be called upon within our code.
